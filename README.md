# DSA-AI-Coder 🤖

Este repositório contém o desenvolvimento do **Estudo de Caso 1: Assistente de Inteligência Artificial** do curso de Fundamentos de Linguagem Python para Análise de Dados do **Data Science Academy (DSA)**.

## 📋 Sobre o Projeto
O objetivo deste Projeto é construir um assistente virtual baseado em Inteligência Artificial utilizando a linguagem Python. 
O assistente conta com uma interface interativa onde o usuário pode conversar com a IA em tempo real.

## 🚀 Tecnologias e Ferramentas Utilizadas
* **Python** (Linguagem principal)
* **Streamlit** (Interface gráfica web e interativa do assistente)
* **Groq** (API de alta velocidade para processamento do modelo de linguagem de IA)
* **Bibliotecas Python** (`streamlit`, `groq` e dependências integradas)

## 📁 Estrutura de Arquivos
* `dsa_assistente-checkpoint.py`: Código-fonte principal do assistente de IA.

## 🔑 Como Usar e Configurar a API Key
Para que o assistente funcione e responda às suas perguntas, é necessário uma chave de acesso (API Key) da Groq. 

Siga os passos:

1. Acesse o site [Groq Console](https://console.groq.com/keys) e crie uma conta gratuita.
2. Gere uma nova chave em **API Keys**.
3. Ao rodar o projeto, cole a chave gerada no campo **"Insira sua API Key Groq"** na barra lateral da interface.
